#markdown语法
###特殊字符
    '<' 替换 &lt、'&' 替换 &amp
###标题
    '#' ------------------- 一级标题h1
    '##' ------------------- 二级标题h2
    '###' ------------------- 三级标题h3
    可以增加到6个'#'
###字体样式
    \*\***加粗**\*\*、\**斜体*\*、\~~~~删除线~~\~~
###列表
    无序列表使用'*'、'+'或'-'标记
    有序列表前面加数字
###引用
    '>' ------------------- 引用内容
>这里是引用内容

###分页线
    '-------'
###链接
    语法：
    这里是链接[链接文本]
    [链接文本]: 链接地址
这里是链接[1]
这里是链接[2]
[1]: fddsg
[2]: sfgsd
###图片
    语法：
    ![Alt text](/path/to/img.jpg)
    ![Alt text](/path/to/img.jpg "Optional title")
###分段和换行
    分段：两个或多个回车
    换行：行末两个空格，回车\
###代码
    <pre><code>代码</pre></code>
    行内代码：`行内代码用反引号`
eg:Use the `printf()` function.
